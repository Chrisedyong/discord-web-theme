Discord Background Color Only

Files:
- manifest.json
- styles.css

Install:
1. Unzip this folder.
2. Open chrome://extensions
3. Turn on Developer mode.
4. Click Load unpacked.
5. Select this folder.
6. Refresh discord.com/app

To change the color, edit this line in styles.css:
--discord-bg-color-only: #1b2f5b;
